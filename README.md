# Path Tracing Demo

### Clone Repository
```bash
git clone --recursive https://github.com/marcushcarter/rust-opengl.git RustRaytracing
cd RustRaytracing
git submodule update --init --recursive
```

**Build & Run**
```bash
# Compile with rust (cargo rustup)
cargo run
```

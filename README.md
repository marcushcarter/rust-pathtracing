### Clone Repository
```bash
git clone --recursive https://github.com/marcushcarter/rust-pathtracing.git rust-pathtracing
cd rust-pathtracing
git submodule update --init --recursive
```

**Build & Run**
```bash
# Compile with rust (cargo rustup)
cargo run
```
